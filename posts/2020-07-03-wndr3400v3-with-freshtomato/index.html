wndr3400v3 Open Source Firmware: freshtomato

January 3, 2020

For a while I was looking for a good open source firmware to run on my old WNDR3400v3. And I finally found it in freshtomato

The upgrade was easy. I just uploaded this image (unzipped). I used the ipv6 since I do not need vpn capabilities for the build and am running ipv6 inhouse.

Afterwards, I set it up as a wireless repeater using this guide. Works like a charm as a repeater for my Ubiquity AC-Pro.

Horray for open source!
