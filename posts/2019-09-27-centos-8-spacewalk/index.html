CentOS 8 and Spacewalk 2.9

September 27, 2019

With CentOS 8 released, it is time to start integrating it and see what works and what doesnt! First was spacewalk.

We are running 2.9 and there is no repository for 2.9 for that version. So we have to use their nightly repo. The following changes were needed to get it to work (assuming that the new centos 8 channel/keys/repo are setup in Spacewalk). I would paste the snippets from my faviorite automation tool, but if you can read this you can translate it :) 

dnf config-manager --add-repo https://copr-be.cloud.fedoraproject.org/results/%40spacewalkproject/nightly-client/epel-8-x86_64
dnf -y install rhn-client-tools rhnpush  rhnsd dnf-plugin-spacewalk osad rhncfg spacewalk-oscap rhn-custom-info rhn-setup rhncfg-actions rhn-check
Thats it! Previously, m2crypto was required. That was REMOVED in CentOS 8. I did not have to change any of the other commands to register with spacewalk. Even osad works!
